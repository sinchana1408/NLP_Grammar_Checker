# AI Grammar & Spell Checker

This project is a simple web application that checks and corrects grammar and spelling mistakes in a sentence. The user can enter a sentence with grammatical errors, and the system will return a corrected version of the text.

The application is built using **Python and Flask** for the backend and **HTML, CSS, and JavaScript** for the frontend. A **Transformer-based NLP model** is used to analyze the sentence and generate the corrected output.

The goal of this project is to demonstrate how Natural Language Processing can be used to build a basic writing assistant that helps improve sentence structure and grammar.

## Features

- Detects grammar mistakes in sentences  
- Corrects spelling errors  
- Simple and clean web interface  
- Real-time correction using a Flask backend  

## Technologies Used

- Python  
- Flask  
- HuggingFace Transformers  
- PyTorch  
- HTML  
- CSS  
- JavaScript
